# 🧪 Experiment 10: CRUD Operations using Node.js & Express.js

## 🎯 Objective
To perform Create, Read, Update, Delete (CRUD) operations using Node.js, Express.js, and MongoDB.

---

## 💻 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman

---

## ⚙️ Features Implemented
- Create a student record
- Read all student records
- Read a single student record
- Update student details
- Delete a student record
